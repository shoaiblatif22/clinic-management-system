# Clinical Management System - Roadmap

--------------------------------------

## Project Overview
The Clinical Management System aims to provide a comprehensive platform for managing clinical operations within the diabetic space, especially for type 1 diabetics. The project will employ modern software engineering practices, including CI/CD pipelines, containerization, and scalable deployments.

## Current Focus: Completing user service, login/registration functionality

## Future roadmap:
- Connect dexcom API to backend service
- Create payment platform using stripe
- Create an admin/doctor dashboard to observe patient data
- Create a patient dashboard where they can connect their sensor to web app and observe their data
- Create an AI insights tool using an Open-API wrapper
